Consider the routing protocols given in List I and the names given in List II:
$$\begin{array}{|ll|ll|}\hline & \textbf{List I} &  & \textbf{List II} \\\hline  \text{(i)} & \text{Distance Vector routing} & \text{(a)}& \text{Bellman-Ford} \\\hline   \text{(ii)}& \text{Link state routing} & \text{(b)}  & \text{Dijkstra} \\\hline  \end{array}$$

For matching of items in List I  with those in List II, which ONE of the following options is CORRECT?
  1. (i) - (a) and (ii) - (b)
  2. (i) - (a) and (ii) - (a)
  3. (i) - (b) and (ii) - (a)
  4. (i) - (b) and (ii) - (b)

It’s straightforward that the Distance Vector Routing algorithm (DVR) uses the Bellman-Ford technique, whereas the Link State Routing algorithm uses Dijkstra's algorithm.

Option $(A)$ is correct.

Distance Vector Routing → uses Bellman-Ford Algorithm
→ because each router shares its distance vector (cost to all known destinations) with neighbors, and Bellman-Ford is used to calculate the shortest path in such distributed environments.

Link State Routing → uses Dijkstra Algorithm
→ because every router builds a complete map of the network (link state database) and independently computes the shortest path tree using Dijkstra’s algorithm.

                               A. (i) - (a) and (ii) - (b)  So, Option (A) is correct
