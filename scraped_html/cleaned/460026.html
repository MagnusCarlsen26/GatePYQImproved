We are given a graph with edges:
$AC = 1$, $BD = 3$, $AD = 6$, $AB = 7$, $DC = 8$, and $BC = x$.

We seek the maximum $x$ such that edge $BC$ is in every MST.

In Kruskal’s algorithm, after adding edges of weight $< x$, if $B$ and $C$ are in separate components and no other edge of weight $\le x$ connects them, then $BC$ must be in every MST.

For $x = 5$: edges with weight $< 5$ are $AC = 1$ and $BD = 3$, yielding components ${A,C}$ and ${B,D}$.
The only edge of weight $\le 5$ connecting these is $BC = 5$.

  • $BC$ is forced into the MST (unique MST of weight $1+3+5=9$).

For $x = 6$: both $BC = 6$ and $AD = 6$ connect the same components.

  • Two MSTs exist (one with $BC$, one with $AD$).
  • $BC$ is not in every MST.

Thus, the largest $x$ for which $BC$ appears in every MST is:

$$
\color{skyblue} \boxed{\text{Answer: 5}}
$$

Answer 5

For MST(minimum spanning tree) we start with min weight edges and we make sure that no cycle is formed before selecting every min edge  . So , firstly we fix edge (b,c) as per condition given in the question.Now ,  edge(d,b)=1 is selected since least weight then edge(a,c)=3 . Now if select one  more edge it result in cycle which is undesirable condition for MST.

So, the only posible values for x is 4,5 (because value of x  should be less than 6). The value of x should be less than 6 because if it is more than 6 edge(a,d) will be selected which evolves the graph getting a cycle .In the question they asked for max posible value therefore ans is 5.
