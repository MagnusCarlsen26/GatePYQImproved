​​​​Consider a binary tree $T$ in which every node has either zero or two children. Let $n>0$ be the number of nodes in $T$.

Which ONE of the following is the number of nodes in $T$ that have exactly two children?
  1. $\frac{n-2}{2} $
  2. $\frac{n-1}{2}$
  3. $\frac{n}{2}$
  4. $\frac{n+1}{2}$

  • The degree of a vertex = number of edges touching it.
  • A vertex is a leaf if and only if it's degree is 1.
  • and Except for root, the two-children nodes have degree 3.


from graph theory we know :
For any tree:  \(|E| = |V| - 1.\)
For any graph:  \(2|E| = \sum_{v \in V} \deg(v).\)

\(\Rightarrow\) n is total number of nodes given in question
\(\Rightarrow\) E = n-1
\(\Rightarrow\) 2(n-1) = 3x+1y-1

  • ( Here x is two children nodes , y is 0 children node (leaf node) except root node so -1 ; as it is included already in 3x )

\[
\Rightarrow\; n = x + y
\]

\[
\Rightarrow\; 2n - 2 = 2x + x + y - 1 \qquad (n = x + y)
\]

\[
\Rightarrow\; 2n - 2 = 2x + n - 1
\]

\[
\Rightarrow\; 2n - n - 2 + 1 = 2x
\]

\[
\Rightarrow\; n - 1 = 2x
\]

\[
\Rightarrow\; x = \frac{n - 1}{2}
\]

\[\text{So answer B.}\]

Question is saying about Full binary tree

For any binary tree

Number of leaf nodes = number of nodes with 2 children(degree 2) + 1

In case of Full binary tree: internal nodes = degree 2 nodes

number of leaves(L) = number of internal nodes(i) + 1

total nodes = n

=> n = i + L

n = i + (i+1)

n = 2i +1

so number of internal nodes = (n-1)/2

It's a FULL Binary Tree

So,

  • Leaf nodes → have 0 children

  • Internal nodes → always have 2 children.

No. of Leaf nodes is given by => L = m+1 ( m = no. of nodes of degree 2) (Standard Property)

Here all internal nodes have a degree 2 => m = I ( I = no. of Internal nodes of tree)

=> L = I+1

Total nodes of tree = n

n = I + L                     ( Total nodes = Internal nodes + Leaf nodes )

=> n = I + (I+1) (From above)

=> n = 2I + 1

=> I = n-1/2                 (i.e The no. of internal nodes (or) no. of nodes of degree 2 )

That's Option B.

Observe that in a Binary tree in which nodes have either 0 or 2 children, the leaf nodes have exactly 0 children and the internal nodes have exactly 2 children. Hence the question is asking for the sum of the internal nodes.

From the leaf internal theorem we know that number of leaf nodes = number of internal nodes + 1

leaf nodes + internal nodes = 2*internal nodes + 1 = Total nodes = n

thus internal nodes = $(n-1)/2$

\[
\begin{aligned}
&\text{In a full binary tree (each node has either 0 or 2 children):} \\[6pt]
&\text{Let } i = \text{ number of internal nodes (with 2 children)}, \\
&\quad\;\; l = \text{ number of leaves (with 0 children)}. \\[6pt]
&\text{We know that } l = i + 1. \\[6pt]
&\text{So, total no. of nodes (n) =  }     i + l \\
&\quad = i + (i+1) \\
&\quad = 2i + 1. \\[6pt]
&\implies i = \dfrac{n-1}{2}.
\end{aligned}\]
$\text{So,The Correct is answer : Option B) } \; {\dfrac{n-1}{2}}$
