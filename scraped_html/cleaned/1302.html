d) Access right information --> YES

Typical rights:

  • R → Read

  • W → Write

  • X → Execute

  • Valid/Invalid bit

  • Dirty bit

  • Referenced bit

These control how the page can be used.

✔️ Correct Answer

The essential content(s) in each page table entry are:

✔️ Page frame number (PFN)

✔️ Access right information

So correct option:


(d) Access right information is essential
(b) PFN is also essential but not listed as the ONLY correct answer.

If the question expects only one option , usually the answer is:

(b) Page frame number

because it is the MOST essential mapping information.
