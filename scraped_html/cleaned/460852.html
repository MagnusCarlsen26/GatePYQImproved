Consider two grammars $G_{1}$ and $G_{2}$ with the production rules given below:


$G_{1} : S \rightarrow$ $if$ $E$ $then$ $S$ $|$ $if$ $E$ $then$ $S$ $else$ $S$ $|$ $a$

$E \rightarrow b$

$G_{2} :S \rightarrow$ $if$ $E$ $then$ $S$ $|$ $M$

$M \rightarrow$ $if$ $E$ $then$ $M$ $else$ $S$ $|$ $c$

$E \rightarrow b$

where $if, then, else, a, b, c$  are the terminals.

Which of the following option(s) is/are CORRECT?

  1. $G_{1}$ is not $LL(1)$ and $G_{2}$ is $LL(1)$
  2. $G_{1}$ is $LL(1)$ and $G_{2}$ is not $LL(1)$
  3. $G_{1}$ and $G_{2}$ are not $LL(1)$
  4. $G_{1}$ and $G_{2}$ are ambiguous.

We are given two context-free grammars $ G_1 $ and $ G_2 $ over the terminal alphabet $\{ \text{if}, \text{then}, \text{else}, a, b, c \}$:

$$
\begin{aligned}
G_1: \quad & S \to \text{if } E \text{ then } S \mid \text{if } E \text{ then } S \text{ else } S \mid a \\
& E \to b \\
G_2: \quad & S \to \text{if } E \text{ then } S \mid M \\
& M \to \text{if } E \text{ then } M \text{ else } S \mid c \\
& E \to b
\end{aligned}
$$

Ambiguity Analysis

A grammar is ambiguous if there exists at least one string in its language that has two distinct leftmost derivations (LMDs) or two distinct parse trees.

Grammar $ G_1 $

Consider the string
$$
w_1 = \text{if } b \text{ then if } b \text{ then } a \text{ else } a
$$

Leftmost Derivation 1 (else binds to inner if):
$$
\begin{aligned}
S &\Rightarrow \text{if } E \text{ then } S \\
&\Rightarrow \text{if } b \text{ then } S \\
&\Rightarrow \text{if } b \text{ then } (\text{if } E \text{ then } S \text{ else } S) \\
&\Rightarrow \text{if } b \text{ then } (\text{if } b \text{ then } a \text{ else } a)
\end{aligned}
$$

Leftmost Derivation 2 (else binds to outer if):
$$
\begin{aligned}
S &\Rightarrow \text{if } E \text{ then } S \text{ else } S \\
&\Rightarrow \text{if } b \text{ then } S \text{ else } S \\
&\Rightarrow \text{if } b \text{ then } (\text{if } E \text{ then } S) \text{ else } a \\
&\Rightarrow \text{if } b \text{ then } (\text{if } b \text{ then } a) \text{ else } a
\end{aligned}
$$

The two derivations correspond to different parse trees (different association of the `else` clause). Hence, $ G_1 $ is ambiguous.

Grammar $ G_2 $

Consider the string
$$
w = \text{if } b \text{ then if } b \text{ then } c \text{ else if } b \text{ then } c \text{ else } c
$$

Leftmost Derivation 1:
$$
\begin{aligned}
S &\Rightarrow \text{if } E \text{ then } S \\
&\Rightarrow \text{if } b \text{ then } S \\
&\Rightarrow \text{if } b \text{ then } M \\
&\Rightarrow \text{if } b \text{ then } \text{if } E \text{ then } M \text{ else } S \\
&\Rightarrow \text{if } b \text{ then } \text{if } b \text{ then } c \text{ else } S \\
&\Rightarrow \text{if } b \text{ then } \text{if } b \text{ then } c \text{ else } M \\
&\Rightarrow \text{if } b \text{ then } \text{if } b \text{ then } c \text{ else } \text{if } E \text{ then } M \text{ else } S \\
&\Rightarrow \text{if } b \text{ then } \text{if } b \text{ then } c \text{ else } \text{if } b \text{ then } c \text{ else } c
\end{aligned}
$$

Leftmost Derivation 2:
$$
\begin{aligned}
S &\Rightarrow M \\
&\Rightarrow \text{if } E \text{ then } M \text{ else } S \\
&\Rightarrow \text{if } b \text{ then } M \text{ else } S \\
&\Rightarrow \text{if } b \text{ then } \text{if } E \text{ then } M \text{ else } S \text{ else } S \\
&\Rightarrow \text{if } b \text{ then } \text{if } b \text{ then } c \text{ else } S \text{ else } S \\
&\Rightarrow \text{if } b \text{ then } \text{if } b \text{ then } c \text{ else } \text{if } E \text{ then } S \text{ else } S \\
&\Rightarrow \text{if } b \text{ then } \text{if } b \text{ then } c \text{ else } \text{if } b \text{ then } M \text{ else } S \\
&\Rightarrow \text{if } b \text{ then } \text{if } b \text{ then } c \text{ else } \text{if } b \text{ then } c \text{ else } c
\end{aligned}
$$

Both derivations produce the same string $ w $, but the parse trees differ:

  • In Derivation 1, the first else binds to the innermost if.
  • In Derivation 2, the first else binds to the middle if, and the inner if is unmatched inside the else branch.

Since the nonterminal $ S $ can appear in the else clause of $ M $, and $ S $ can generate an unmatched if, the grammar permits multiple structural interpretations of the same string. Thus, $ G_2 $ is ambiguous.

LL(1) Property

A grammar is LL(1) only if it is unambiguous, has no left recursion, and for every nonterminal $ A $ with alternative productions $ A \to \alpha_1 \mid \alpha_2 \mid \dots $, the sets $ \text{FIRST}(\alpha_i) $ are pairwise disjoint (with appropriate handling of $ \varepsilon $-productions via FOLLOW sets).

Crucially, any ambiguous grammar cannot be LL(1), because an LL(1) parser must make a unique parsing decision using only one token of lookahead. Since both $ G_1 $ and $ G_2 $ are ambiguous, neither can be LL(1). Moreover, both grammars exhibit FIRST/FIRST conflicts:

  • In $ G_1 $, the productions $ S \to \text{if } E \text{ then } S $ and $ S \to \text{if } E \text{ then } S \text{ else } S $ both begin with `if`, so their FIRST sets overlap.
  • In $ G_2 $, the productions $ S \to \text{if } E \text{ then } S $ and $ S \to M $ both can derive strings starting with `if` (since $ M \Rightarrow \text{if } E \text{ then } \dots $), leading to overlapping FIRST sets for $ S $.

Thus, both grammars violate the fundamental requirement for LL(1) parsing.

  • $ G_1 $ is ambiguous and not LL(1).
  • $ G_2 $ is ambiguous and not LL(1).

Therefore, the correct statements are:

$$
\color{gold }\boxed{\text{Answer: C, D}}
$$
