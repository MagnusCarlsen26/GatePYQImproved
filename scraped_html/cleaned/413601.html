Given the following relation instances $$\begin{array}{|ll|ll||ll|}\hline & \textbf{X} &  & \textbf{Y } &  \textbf{ Z} \\\hline & \text{1} & & \text{4} & \text{ 2} \\\hline   & \text{1} & & \text{5} & \text{ 3} \\\hline  & \text{1} & & \text{4} & \text{ 3} \\\hline & \text{1} & & \text{5} & \text{ 2} \\\hline & \text{3} & & \text{2} & \text{  1}& \\\hline  \end{array}$$

Which of the following conditions is/are TRUE ?

  1. $\mathrm{XY} -> \mathrm{Z}$ and $\mathrm{Z} -> \mathrm{Y}$
  2. $\mathrm{YZ} -> \mathrm{X}$ and $\mathrm{X} -> >\mathrm{Y}$
  3. $\mathrm{Y} -> \mathrm{X}$ and $\mathrm{Y} ->>\mathrm{X}$
  4. $\mathrm{XZ}->\mathrm{Y}$ and $\mathrm{Y}->\mathrm{X}$

First lets assume we are only interested in the given instance. (Based on an instance we can only reject FDs but can't guarantee that they will be valid throughout the database.) For this question, let's solve for the given instance.

Option A:
The depencency Z -> Y has a violation (tuples 2,3). So this option is false.

Option B:
The first FD is a valid. Fot the Multivalued Dependency let's use the formal definition:


Considering the first 4 tuples, this follows exactly the template above, so this is a valid MVD.
Hence option B is correct.

Option C:
Y -> X is valid.
An ordinary functional dependency between 2 attributes imples MVD.

Hence Y->>X is also valid.
Hence option C is correct.

Option D:
The first FD has a violation (can be easily checked, see tuples 2 and 3)
So option D is false.

Therefore, answer is: B and C are true.


P.S.

A relation R is in Fourth Normal Form (4NF) if:

- R is in BCNF (Boyce-Codd Normal Form).
- R does not contain any multivalued dependencies.

Source: Multivalued dependency , Stanford Slides

FD: One X value determines exactly one Y value, e.g., X → 1 (X determines only 1)
MVD: One X value determines a set of Y values independent of other attributes, e.g., X →> 1,2 (X can determine multiple values 1 and 2)
