Consider the following two languages over the alphabet $\{a, b\}$:

$$\begin{array}{l} L_{1}=\left\{\alpha \beta \alpha \mid \alpha \in\{a, b\}^{+} \text {AND } \beta \in\{a, b\}^{+}\right\} \\ L_{2}=\left\{\alpha \beta \alpha \mid \alpha \in\{a\}^{+} \text {AND } \beta \in\{a, b\}^{+}\right\}
\end{array}$$

Which ONE of the following statements is CORRECT?

  1. Both $L_{1}$ and $L_{2}$ are regular languages.
  2. $L_{1}$ is a regular language but $L_{2}$ is not a regular language.
  3. $L_{1}$ is not a regular language but $L_{2}$ is a regular language.
  4. Neither $L_{1}$ nor $L_{2}$ is a regular language.

We are given two languages over the alphabet $\Sigma = {a, b}$:

$$
L_1 = { \alpha \beta \alpha \mid \alpha \in {a,b}^+, ; \beta \in {a,b}^+ }
$$
$$
L_2 = { \alpha \beta \alpha \mid \alpha \in {a}^+, ; \beta \in {a,b}^+ }
$$

Analysis of $L_1$

The language $L_1$ consists of all strings that can be written as $\alpha \beta \alpha$, where both $\alpha$ and $\beta$ are nonempty strings over ${a, b}$. Hence, the prefix and suffix must be identical and nonempty.

To verify this property, a finite automaton would need to “remember” an arbitrary-length prefix $\alpha$ to compare it with the suffix, which requires unbounded memory. This strongly suggests non-regularity.

Proof using the Myhill–Nerode Theorem

A language $L$ is regular if and only if its Myhill–Nerode relation $\equiv_L$ has finitely many equivalence classes. Two strings $x, y$ are distinguishable with respect to $L$ if there exists some string $z$ such that exactly one of $xz$ or $yz$ is in $L$.

Consider the infinite set
$$
S = { a^n \mid n \ge 1 }.
$$
We show that any two distinct elements $a^i, a^j$ with $i \ne j$ are distinguishable with respect to $L_1$.

Let $i < j$ and take $z = b a^i$. Then:

  • $a^i z = a^i b a^i \in L_1$, since choosing $\alpha = a^i$ and $\beta = b$ gives $\alpha \beta \alpha = a^i b a^i$.

  • $a^j z = a^j b a^i \notin L_1$, because in any decomposition $\alpha \beta \alpha$, the prefix and suffix must be equal, but here the prefix is $a^j$ and the suffix $a^i$, which differ since $i \ne j$.

Hence, all $a^n$ are in distinct equivalence classes under $\equiv_{L_1}$, and therefore $L_1$ has infinitely many equivalence classes.

Thus, $L_1$ is not regular .

Analysis of $L_2$

In $L_2$, both $\alpha$ and $\beta$ are nonempty, but $\alpha$ is restricted to ${a}^+$, i.e., $\alpha = a^k$ for some $k \ge 1$.

Let us observe what strings belong to $L_2$.

Any string $w \in L_2$ satisfies:
$$
w = a^k , \beta , a^k,
$$
for some $k \ge 1$ and $\beta \in {a,b}^+$.

Thus, every such $w$:

  • starts with $a$,

  • ends with $a$, and

  • has length at least 3 (since $\beta \ne \varepsilon$ and $k \ge 1$).

Conversely, if a string $w$ over ${a,b}$ starts and ends with $a$ and has at least one symbol in between, then we can pick $\alpha = a$ and $\beta$ as the substring between the first and last $a$.
Hence $w = a , \beta , a$, with $\alpha = a \in {a}^+$ and $\beta \in {a,b}^+$.

Therefore,
$$
L_2 = { w \in {a,b}^* \mid w \text{ starts with } a,; w \text{ ends with } a,; |w| \ge 3 }.
$$

Regularity of $L_2$

The language $L_2$ can be described by the regular expression
$$
a (a + b)^+ a.
$$
This clearly defines a regular language, since the class of regular languages is closed under concatenation and the Kleene plus operation.

A simple DFA can recognize $L_2$ by:

  1. Checking that the first symbol is $a$,

  2. Reading one or more middle symbols,

  3. Accepting only if the final symbol is $a$.

Thus, $L_2$ is regular .

$$
\color{lime} \boxed{\text{C. } L_1 \text{ is not regular but } L_2 \text{ is regular.}}
$$

L1 = { αβα | α ∈ {a, b}+ AND β ∈ {a, b}+ }

Here, α appears twice in the string, meaning the language requires matching substrings .This kind of pattern cannot be expressed using a finite-state automaton (DFA/NFA), as regular languages cannot enforce constraints on repeated substrings .

L1 is NOT regular .

L2 = { αβα | α ∈ {a}+ AND β ∈ {a, b}+ }

For this Language we can create regular expression a+(a,b)+a+ that's mean starting and ending with a.

L2 is  regular .

L1 is Non-Regular

Method I
Using MyHill Nerode Theorem, you can create an infinite set of distinguishable strings. Such a set S is {a^n | n>=1}. Now for any two strings a^x and a^y (x != y), you can have an extension (say, ba^x) which leads a^x to the accepting state but a^y to a non-accepting state. Thus, we say that L1 is non-regular with the help of MyHill Nerode Theorem.


Method II
Infinite possible values of alpha and beta to keep track of along with comparison among them (matching the LHS and  RHS of beta ) => Non-Regular Language.

L2 is Regular

L2 can be reduced to the language: "Minimum length of string = 3. Start and end with 'a' & all the symbols in between are absorbed by beta ". I mean to say,  w = abbaaaba can be reduced to ( alpha = a ) and ( beta = bbaaab ), thus, w belongs to L2.
